    Windows W32tm ConfigurationLink

    Enable Windows Time service (W32Time) Services.

    Service for various other servers and devices on your network, if you are behind a firewall or proxy you will need to allow port 123 UDP out from your network.

    Some DSL routers may act as your default time source, if that is the case this must disabled.

    If you running or using a virtual enviroment please note they may orrerride these changes, so this disable for the devices which will act as your sources.

    Check Current ConfigurationLink

     w32tm /query /configuration

 w32tm /query /peers

    Cause a RysncLink

    w32tm /resync /nowait

    Starting and stopping Time ServicesLink

    There can be a delay when doing this.Open a Windows Command prompt.

    net stop w32time

net start w32time

    Using External NTP ServersLink

    Depending on which part of world you are in use the nearest NTP source.

    Some times its best to use your ISP servers. Check there support sites.

    Running these two commands will give you the current settings.

     W32tm /query /configuration

 w32tm /tz Display the current time zone settings

    Syncing with external servers.

     w32tm /config /update /manualpeerlist:"0.pool.ntp.org,0x8 1.pool.ntp.org,0x8 2.pool.ntp.org,0x8 3.pool.ntp.org,0x8" /syncfromflags:MANUAL

    Check current New NTP configurationLink

    w32tm /query /configuration

w32tm /query /status

    There may be a delay before changes take affect.
