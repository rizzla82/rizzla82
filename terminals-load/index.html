    Load multiple TerminalsLink

    Enable Tabs support for your terminal.

    Each application or service you want to load must be already installed.

     sudo -i

 apt install htop
 apt install sensors
 apt install iftop

    Docker is installed separately or another service you want to monitor.

    Create script using a text editor and once saved make as script.

    Save as apps_load.shLink

    #!/bin/sh
gnome-terminal --app-id us.kirkland.terminals.byobu -e byobu \
--tab -e "htop" \
--tab -e "watch sensors" \
--tab -e "watch docker ps -a" \
--tab -e "sudo iftop" \

    The sudo command will require you to enter the password, click tab.

    Make the saved file executable.Link

    chmod u+x apps_load.sh

    Run from terminal or double and run in terminalLink

    ./apps_load.sh
